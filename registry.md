# Container registry

```yaml
  - online storage space for docker image
  - like docker hub
```

# Artifact registry

```yaml
  - the next level of container registry
  - use can store docker image
  - and java (maven package), nodejs (npm package) or golang package or ...
  - pricing based on GCS
```
